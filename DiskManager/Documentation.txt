Project: System Monitoring for Linux Machine

Features: 
	1.) Monitors available disk space and free space
	2.) Displays total numbers of mounted drives
	3.) Check Memory Usage

Linux Commands Used:
	1.) df command - Report file system disk space usage
	2.) free command - Display amount of free and used memory in the system
